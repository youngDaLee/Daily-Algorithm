### 접근 방법을 적용한 코드
```
def solution(prices):
    answer = []
    stack = []
    for i in range(0, len(prices)):
        cnt = 0
        stack.append(prices[i])
        for j in range(i, len(prices)):
            if stack[-1] > prices[j] or j+1==len(prices) :
                answer.append(cnt)
                break
            cnt = cnt+1

    return answer
```
